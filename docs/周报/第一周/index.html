第一周

创建时间：2021年12月13日 10:00

别做井底之蛙。

  1. 字母’x’在CSS世界中的角色和故事

一篇介绍关于在 css 世界中 baseline 的文章，讲得很不错。也可以看这个博主的其他相关文章，写得也很好。

  1. Swift 中文官网 SwiftGG

我打算学习 ios 开发，虽然其他的也没精通，但是我想真正的打通大前端，web （vue，react），微信小程序，nodejs，react-native，android，ios，flutter。

  1. css 属性快捷查找

这个网站方便快速查询对应的属性，在开发过程中查看很方便，当然还有一种方式就是在浏览器中搜索指定属性像下面这样 css line-height mdn ，里面对属性的说明更详细一些。

  1. Detailed Guide on Android Clean Architecture

一篇讲最佳 android clean 架构的文章。目前我的水平还不能全部理解，但是觉得有道理。

  1. How To Set Up a Dependency Injection Framework With Dagger2 in Android

讲 dagger2 在 android 中使用的文章，优秀的文章，虽然现在 android 推荐使用 Hilt ，但是学习 dagger2 还是有必要的，最近我就在学习这个。

  1. 我私藏的10个神级编程资源，远超培训机构！程序员免费学习资源整合推荐，Java/算法/前端/Python/Go/GitHub

这是 B站 UP 主程序员鱼皮 的视频，里面推荐了几大编程资源，我觉得都非常不错。

  • JavaGuide 「Java学习+面试指南」一份涵盖大部分 Java 程序员所需要掌握的核心知识。准备 Java 面试，首选 JavaGuide！
  • leetcode 学习算法和刷面试题的地方，非常好的网站，包含的测试用例覆盖面很广，是我知道的几个中最好的一个。
  • 被删的前端游乐场 前端的技术分享。
  • Python - 100天从新手到大师 这是一个分阶段的 Python 入门教程，爬虫，数据分析等等。
  • GitHub 漫游指南 讲 github 的。
  • sql 自学网 xuesql.cn,一个适合小白学SQL的网站，我们会由浅入深的介绍所有有关 SQL 的知识, 每一个章节是一组相关的SQL知识点; 并且会配备一组动手练习任务。
  • docker 从入门到实践 本书既适用于具备基础 Linux 知识的 Docker 初学者。
  • 《动手学深度学习》 这是这本书对应的网站，讲得很详细。
  • 地鼠文档 这是关于 go 语言学习的。

当然还有很多优秀的资源在 UP 主的编程导航中发现。
